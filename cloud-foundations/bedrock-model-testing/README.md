#Amazon Bedrock Model Testing

## Objective
Explore Amazon Bedrock by testing available foundation models and understanding how prompts and changing configurations affect responses.

## Tools & Services
- Amazon Bedrock
- AWS Management Console

## What I Did
- Accessed Amazon Bedrock via the AWS Console
- Tested different foundation models
- Observed differences in output based on the same prompt and different settings

## Key Takeaways
- Prompt wording significantly impacts model responses
- Configuring Top P level and Temperature significantly affects models responses
- Bedrock provides a simple interface for experimenting with foundation models
